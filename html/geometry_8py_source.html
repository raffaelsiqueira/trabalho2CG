263  p2 = self.p2

265  p4 = that.p2

269  return None

272  p43 = p4 - p3
273  p21 = p2 - p1

277  d1321 = p13.x * p21.x + p13.y * p21.y + p13.z * p21.z

463 ## Calculates the area of a planar polygon.
464 # The algorithm is as follows:<br>
465 #
466 # Traverse the loop of coordinates, assuming that it is in
467 # clockwise order, computing the components of the "area" of the
468 # enclosed polygon. The total "area" components are computed by
469 # adding "area" components (cross product components) of
470 # triangles sides formed by the first, previous, and current
471 # vertices. If the loop is not convex, some of the triangle
472 # areas might be negative, but those will be compensated by other
473 # positive triangle areas so that the final area is positive.<br>
474 #
475 # Note: area here is actually twice the area. <br>
476 # positive here means in the direction of the face normal.
477 #
478 # @return twice the polygon area.
479 #
